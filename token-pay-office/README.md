## Описание / Description

Token Pay Office — это концепция физических и виртуальных офисов обслуживания TOKEN PAY LLC, предоставляющих:

- **Верификацию личности** — личная верификация аккаунтов TOKEN PAY ID
- **Поддержку клиентов** — помощь с настройкой VPN, оплатой, восстановлением доступа
- **Корпоративные услуги** — интеграция предприятий с TOKEN PAY ID, подписание договоров
- **Финансовые операции** — приём оплаты, выдача промокодов, управление подписками

## Интеграция

Token Pay Office интегрируется с TOKEN PAY ID через стандартный OAuth 2.0 + PKCE, как любое другое предприятие:

- `client_id` для Office будет зарегистрирован как enterprise-приложение
- Использует `/api/v1/oauth/authorize` → `/api/v1/oauth/token` flow
- Доступ к `/api/v1/enterprise/sync-user` и `/api/v1/enterprise/user-data`
- Webhook-уведомления через стандартную систему webhooks
