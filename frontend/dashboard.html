Секрет подписи

Каждый запрос содержит заголовок X-TPID-Signature для проверки подлинности.

не установлен
Как проверить подпись в коде
Node.js / Express
const crypto = require('crypto');

app.post('/webhooks/tpid', express.json(), (req, res) => {
  const sig  = req.headers['x-tpid-signature'];
  const body = JSON.stringify(req.body);
  const hmac = crypto.createHmac('sha256', YOUR_SECRET)
                     .update(body).digest('hex');
  if (sig !== hmac) return res.status(401).end();

  const { event, data } = req.body;
  // handle event...
  res.json({ ok: true });
});

Как это работает: 1) Нажмите «Создать QR» 2) Откройте камеру на телефоне 3) Наведите камеру на QR-код 4) Перейдите по открывшейся ссылке — вы войдёте автоматически

На странице входа другого устройства отображается QR-код? Отсканируйте его камерой или вставьте ссылку ниже, чтобы подтвердить вход.

Как это работает: 1) На другом устройстве откройте страницу входа 2) Нажмите «Вход по QR-коду» 3) Здесь нажмите «Сканировать камерой» или скопируйте ссылку из QR и вставьте выше 4) Вход на том устройстве будет подтверждён
